# 4017 Sequencer

A simple LED sequencing project using the CD4017 decade counter.

This module builds on the 555 timer circuit and uses its output as a clock signal to drive sequential LED patterns.

## Objective

The goal of this project is to understand:

- How the CD4017 counts clock pulses
- How sequential digital outputs work
- How to combine a timer circuit with a counter circuit

## Hardware

- CD4017 Decade Counter
- NE555 Timer (used as clock source)
- LEDs
- Resistors
- Breadboard
- 5V power supply
