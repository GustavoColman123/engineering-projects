# 4017 LED Sequencer

This project implements a sequential LED pattern using the CD4017 decade counter.  
The CD4017 is driven by a clock signal generated by a NE555 timer configured in astable mode.

## Objective

The objectives of this module are:

- Understand how the CD4017 counts clock pulses.
- Explore sequential digital outputs.
- Integrate analog timing (555) with digital logic (4017).
- Build a visually demonstrative LED sequencing system.

## System Overview

- **NE555 Timer:** Generates periodic clock pulses.
- **CD4017 Counter:** Advances one output (Q0–Q9) per clock pulse.
- **LEDs:** Visualize the counting sequence.

## Hardware

- CD4017 Decade Counter
- NE555 Timer (clock source)
- LEDs (4–10 units)
- Resistors (220Ω–470Ω for LEDs)
- Breadboard and jumper wires
- 5V power supply
- Capacitors (as required for the 555 timer)

## Pin Connections (Summary)

| Component | Pin | Connection |
|----------|-----|------------|
| CD4017 | 16 | +5V |
| CD4017 | 8 | GND |
| CD4017 | 14 | Clock input from NE555 pin 3 |
| CD4017 | 13 | GND (Clock Enable) |
| CD4017 | 15 | GND (Reset, unless used for limiting sequence) |

## Expected Behavior

Each pulse from the 555 timer advances the CD4017 to the next output, turning on LEDs sequentially. After the last output, the sequence restarts.
